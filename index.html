Projeto de Destaque

BlockMan - Jogo de Plataforma

Uma recriação de um clássico jogo de plataforma com o personagem BlockMan, navegando por níveis com obstáculos e inimigos. Desenvolvido com HTML, CSS e JavaScript, oferece jogabilidade envolvente e gráficos simples.

Ver Projeto

Recursos Implementados:

  • Movimento: Controle via teclas ou gamepad.
  • Moedas: Colete para ganhar pontos.
  • Inimigos: Evite para manter vidas.
  • Níveis: Dificuldade progressiva.
  • Som: Trilha e efeitos sonoros.

Tecnologias:

  • HTML5
  • CSS3
  • JavaScript

Conclusão: Um projeto que une diversão e tecnologia web.
